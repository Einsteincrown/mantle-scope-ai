# MantleScope

**Research Smarter. Trade Safer. Powered by AI on Mantle.**

MantleScope is an AI-powered on chain research agent built for the Mantle 
ecosystem. It lets anyone, traders, researchers, or newcomers — research 
wallets, tokens, and protocols on Mantle using plain English. No technical 
knowledge required.

---

## The Problem

Most people navigate DeFi blind. They can't read raw on-chain data, can't 
tell a risky wallet from a safe one, and have no easy way to research what's 
actually happening on Mantle. So they guess — and guessing is expensive.

## The Solution

MantleScope puts an AI agent in front of that complexity. Ask it anything 
in plain English, it digs into the chain, breaks it down, and gives you 
real answers. 

---

## Features

- **AI Chat Interface** — Ask questions in plain English and get on-chain 
  answers powered by Claude AI
- **Wallet Analysis** — Paste any wallet address and get a full profile 
  with an AI generated risk score from 1–10
- **Token & Protocol Research** — Search any token or DeFi protocol on 
  Mantle and get an AI-generated research summary covering activity, 
  risk factors, and sentiment
- **Ecosystem Explorer** — A living directory of top dApps, protocols, 
  and tokens on Mantle organized by category (DeFi, Gaming, NFT, 
  Infrastructure)

---

## How It Works

1. User asks a question in plain English
2. App fetches raw on-chain data from Mantle via the public RPC endpoint
3. That data is passed to Claude AI alongside the user's question
4. Claude reads and interprets the data and responds in plain English
5. User gets clear, actionable answers instantly

> Mantle supplies the data. Claude supplies the intelligence. 
> MantleScope is the bridge.

---

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Anthropic Claude API
- Mantle Public RPC — `https://rpc.mantle.xyz`
- Deployed on Vercel
